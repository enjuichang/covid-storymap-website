About the Project

Capstone Project Visualization

This project is the product of my graduating Capstone Project for Minerva University. The goal of this project is to show the impact of COVID-19 on human movement (mobility) in Taiwan by using data from transportation statistics, technological companies, and seismic activities. The data was then processed and visualized into story maps for each location.

Please contact enjui.chang@uni.minerva.edu for more information on the project.
